# Contributing Guide

Thank you for your interest in contributing to **AwesomeProject**!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/awesomeproject.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Run tests: `npm test`
6. Submit a pull request

## Code Style

- Follow the existing code style
- Write meaningful commit messages
- Add tests for new features

## Reporting Bugs

Open an issue using the [Bug Report](https://github.com/awesomeorg/awesomeproject/issues/new?template=bug_report.md) template.

## Contact

Reach out to [@leopoldmac](https://github.com/leopoldmac) or the core team with questions.
