# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, email [security@awesomeorg.io](mailto:security@awesomeorg.io) with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect a response within 48 hours. Credits will be given in the release notes.
